# AI Posture Corrector

AI 기반 실시간 자세 감지 및 교정 웹 애플리케이션

## 프로젝트 소개

AI Posture Corrector는 사용자의 웹캠 영상을 실시간으로 분석하여 거북목 자세를 감지하고, 잘못된 자세가 일정 시간 이상 지속될 경우 음성 경고를 제공하는 웹 애플리케이션입니다.

장시간 컴퓨터를 사용하는 대학생, 직장인, 개발자들이 올바른 자세를 유지할 수 있도록 돕는 것을 목표로 합니다.

## 주요 기능

### 실시간 자세 감지

* 웹캠 영상을 실시간으로 분석
* MediaPipe Pose를 활용한 신체 랜드마크 추출
* 자세 변화 감지

### 자세 교정 알림

* 사용자가 설정한 시간 동안 잘못된 자세가 지속될 경우 음성 경고 제공
* 동일 경고 반복 방지를 위한 쿨타임 적용

### 사용자 설정

* 경고 시간 설정

  * 5초
  * 30초
  * 60초

### 통계 기능

* 당일 경고 횟수 기록
* 실시간 자세 상태 표시

## 기술 스택

* HTML5
* CSS3
* JavaScript
* MediaPipe Pose
* Web Speech API

## 시스템 구성

웹캠 입력 → 자세 분석 → 자세 판정 → 음성 경고 → 사용자 교정

## 실행 방법

1. 아래 GitHub Pages 링크에 접속합니다.
2. 브라우저에서 카메라 권한을 허용합니다.
3. 3초 동안 기본 자세를 유지하여 기준 자세를 측정합니다.
4. 실시간 자세 감지 기능을 사용합니다.
5. 잘못된 자세가 설정 시간 이상 지속되면 음성 경고가 제공됩니다.
